## My app 

### How to run the app?
First of all, you must have Xcode installed on your computer.
- close repository
- in console cd to `blockly-swift-ui-generator/MyApp`
- run `pod install`
- move  `AppConfigData.swift` file generated by website into `../MyApp/MyApp/App` and run the code.
